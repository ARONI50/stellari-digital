Stellari LinkOps

Gerador de links inteligentes. A primeira missão é YouTube: criar um link curto da Stellari, exibir prévia para compartilhamento e tentar abrir o vídeo direto no app quando possível.

Observação: a prévia no WhatsApp depende do cache do próprio WhatsApp. O link gerado fica salvo no armazenamento do Netlify.

Como funciona

1
Você cola o linkO sistema identifica o ID do vídeo do YouTube, inclusive Shorts.
2
Ele cria um código curtoExemplo: /l/8bc0y. Cada código aponta para um vídeo diferente.
3
O WhatsApp lê a préviaA página entrega título, descrição e thumbnail por Open Graph.
4
O clique tenta abrir o appA ponte Stellari tenta jogar para o app do YouTube; se falhar, cai no YouTube web.
